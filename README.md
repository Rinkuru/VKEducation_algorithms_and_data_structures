Папка HW_1 - первая домашняя работа, Фундаментальные задачи по массивам
HW_2 - Фундаментальные задачи по связанным спискам, декам и стекам
HW_3 - Фундаментальные задачи по хеш-таблицам, сортировкам

Чтобы собрать и запустить тесты, выполните следующие команды в терминале каждой папки

```bash
cmake -B build
cd build
make
make test
```
