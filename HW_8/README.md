1. Максимальная сумма подмассива
Дан массив целых чисел arr и число k. Требуется найти максимальную сумму среди всех возможных подмассивов длины k.
Реализуйте решение с использованием префиксных сумм и поясните, как оно оптимизирует подсчёт сумм подмассивов.

2. Subarray Sum Equals K
Дан массив целых чисел nums и число k. Найдите количество непрерывных подмассивов, сумма которых равна k.
Используйте префиксные суммы для эффективного подсчёта и проанализируйте сложность алгоритма.

3. Максимальная длина подмассива с равным количеством нулей и единиц
Дан бинарный массив (состоящий из 0 и 1). Найдите длину самого длинного подмассива, в котором количество нулей и единиц одинаково.
Реализуйте решение через преобразование 0 в -1 и использование префиксных сумм.

4. Индекс поворота массива
Дан массив чисел. Необходимо найти индекс, в котором сумма всех элементов слева равна сумме всех элементов справа.
Используйте префиксные суммы или эквивалентный линейный подсчёт для эффективного решения.

5. Баланс скобок через префиксные суммы
Дана строка, состоящая из скобок. Необходимо определить корректность скобочной последовательности с использованием префиксных сумм.
Объясните, как префиксные суммы позволяют отслеживать баланс открывающих и закрывающих скобок и обеспечить линейное время работы алгоритма.
