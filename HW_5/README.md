1. Полное бинарное дерево
Реализуйте функцию, которая проверяет, является ли заданное бинарное дерево полным.
Необходимо корректно определить условие заполненности уровней и объяснить выбранный способ обхода (например, BFS).

2. Объединение K отсортированных массивов
Напишите функцию, объединяющую K отсортированных массивов в один отсортированный массив.
Для эффективного решения используйте мин-кучу и проанализируйте сложность алгоритма.

3. K-ый наименьший / наибольший элемент
Дан массив чисел. Необходимо найти K-ый наименьший или K-ый наибольший элемент.
Рассмотрите эффективные подходы (например, с использованием кучи) и обоснуйте сложность решения.

4. K-ый наименьший / наибольший элемент в BST
Дано бинарное дерево поиска.
Реализуйте алгоритм нахождения K-ого наименьшего или наибольшего элемента, используя свойства BST и обход дерева.

5. Balance factor
На вход функции передаётся указатель на корень бинарного дерева поиска.
Необходимо для каждого узла вычислить и заполнить поле balance_factor (разницу высот левого и правого поддеревьев), объяснив способ вычисления высоты.

6. Преобразование в зеркальное дерево
Реализуйте алгоритм зеркального отражения бинарного дерева, меняя местами левые и правые поддеревья каждого узла.
Поясните, почему алгоритм корректен и какова его временная сложность.
